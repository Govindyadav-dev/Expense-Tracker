# Expense Tracker

A simple Expense Tracker built using HTML, CSS, and JavaScript.  
This project allows users to track their income and expenses easily.

## Features

- Add income and expense transactions
- View total balance
- View total income
- View total expenses
- Simple and clean UI

## Technologies Used

- HTML
- CSS
- JavaScript

expense-tracker

## How to Run

1. Download or clone the repository

git clone https://github.com/your-username/expense-tracker.git

2. Open the folder

3. Run index.html in your browser
